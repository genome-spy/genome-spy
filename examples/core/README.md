# Example Specifications

This directory contains curated shared example specifications that can be
accessed through URLs like `http://localhost:8080/?spec=examples/core/first.json`
when you launch GenomeSpy using the `npm start` command.

If you want to play with specifications but don't want to add them to the
version control, use repo-root `private/` instead. It is in `.gitignore` and is
accessible at `http://localhost:8080/?spec=private/foo.json`.

Legacy `packages/core/private/` is still served temporarily until it is moved
manually.
